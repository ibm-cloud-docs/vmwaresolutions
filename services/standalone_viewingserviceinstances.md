# Viewing on-premises HCX instances

Use this procedure to view the on-premises HCX instances that you ordered for on-premises use and detailed information about them.

## Procedure

1. Click **Deployed Instances** from the left navigation pane.
2. In the **On-premises HCX Instances** table, view the detailed information about the on-premises HCX instances that you ordered:

  * **Name**: The name of the on-premises HCX instance.
  * **Activation key**: The activation key of the on-premises HCX instance.
  * **Creation time**: The date and time that the on-premises HCX instance was created.
  * **Status**: The status of the on-premises HCX instance. The status can have one of the following values:
    <dl class="dl">
    <dt class="dt dlterm">Modifying</dt>
    <dd class="dd">The instance is being created.</dd>
    <dt class="dt dlterm">Installed</dt>
    <dd class="dd">The instance is ready to use.</dd>
    <dt class="dt dlterm">Removing</dt>
    <dd class="dd">The instance is being deleted.</dd>
    </dl>
  * **Actions**: Take the following actions on the on-premises HCX instance:
    <dl class="dl">
    <dt class="dt dlterm">Delete</dt>
    <dd class="dd">Delete the on-premises HCX instance.</dd>
    </dl>

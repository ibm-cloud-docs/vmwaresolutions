# Zerto on IBM Cloud 개요

Zerto on {{site.data.keyword.cloud}} 서비스는 {{site.data.keyword.cloud_notm}}의 VMware virtual 환경에서 데이터를 보호하고 복구하도록 배치 오퍼링으로 통합될 수 있는 복제 및 재해 복구 기능을 제공합니다.

## Zerto on IBM Cloud 서비스의 컴포넌트

다음 컴포넌트가 주문되고 Zerto on {{site.data.keyword.cloud_notm}} 서비스에 포함됩니다.

* Zerto Virtual Replication Appliances의 사용을 위한 {{site.data.keyword.cloud_notm}} 인프라의 사설 포터블 서브넷
* Zerto Virtual Replication이 설치된 Microsoft Windows VSI(Virtual Service Instance)
* 모든 ESXi 서버에 배치되고 구성될 Zerto Virtual Replication Appliances

**참고**: 기본 클러스터에만 ZVM(Zerto Virtual Manager) 컴포넌트가 배치됩니다.

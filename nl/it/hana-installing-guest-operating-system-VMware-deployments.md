# 3. Installazione del tuo SO guest nell'hypervisor ESX (facoltativo)
{: #install_guest_os}

Se decidi di utilizzare l'hypervisor ESX pre-installato, sei responsabile dell'installazione di Red Hat Enterprise Linux per SAP HANA 7.4 OS come sistema operativo guest nell'hypervisor. Devi essere sicuro che vengano soddisfatti i SAP HANA Tailored Data Center Integration (TDI) Key Performance Indicators (KPI) di ogni macchina virtuale su cui viene distribuito SAP HANA.

# 4. Scaricamento e installazione delle applicazioni e del software SAP
{: #install_sap}

Accedi a [SAP Support Portal ![Icona link esterno](../../icons/launch-glyph.svg "Icona link esterno")](https://support.sap.com/en/index.html){: new_window} e fai clic su **Download Software** per le istruzioni di installazione e l'hardware appropriato. Dopo aver installato il tuo hardware e sistema operativo, devi installare SAP HANA. Per le istruzioni di installazione, consulta [*SAP HANA Installation Guides and Implementation Help* ![Icona link esterno](../../icons/launch-glyph.svg "Icona link esterno")](https://www.sap.com/products/hana/implementation/resources.html){: new_window}.

Il tuo software SAP HANA deve essere installato da un programma di installazione SAP HANA certificato che ha completato il corso di certificazione di installazione SAP HANA. Per ulteriori informazioni, vedi [Who Can Install SAP HANA?![Icona link esterno](../../icons/launch-glyph.svg "Icona link esterno")](http://www.saphanacentral.com/p/who-can-install-sap-hana.html){: new_window}.

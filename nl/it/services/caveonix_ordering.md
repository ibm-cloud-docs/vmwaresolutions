# Ordine di Caveonix RiskForesight on IBM Cloud
{: #caveonix_ordering}

Puoi ordinare il servizio Caveonix RiskForesight on {{site.data.keyword.cloud}} quando ordini una nuova istanza VMware vCenter Server con il servizio incluso oppure aggiungendo il servizio alla tua istanza esistente.

## Ordine di Caveonix RiskForesight on IBM Cloud per una nuova istanza
{: #caveonix_ordering-new}

Per ordinare una nuova istanza VMware vCenter Server on {{site.data.keyword.cloud_notm}} con Caveonix RiskForesight on {{site.data.keyword.cloud_notm}}, seleziona **Caveonix RiskForesight on IBM Cloud** nella sezione **Servizi** quando ordini l'istanza dalla console {{site.data.keyword.vmwaresolutions_short}}.

## Ordine di Caveonix RiskForesight on IBM Cloud per un'istanza esistente
{: #caveonix_ordering-existing}

Per aggiungere il servizio Caveonix RiskForesight on {{site.data.keyword.cloud_notm}} in un'istanza VMware vCenter Server on {{site.data.keyword.cloud_notm}} esistente, visualizza l'istanza per cui desideri aggiungere il servizio, fai clic su **Servizi** nel riquadro di navigazione a sinistra e fai quindi clic su **Aggiungi**.

## Configurazione di Caveonix RiskForesight on IBM Cloud
{: #caveonix_ordering-config}

Quando ordini il servizio Caveonix RiskForesight on {{site.data.keyword.cloud_notm}}, fornisci le seguenti impostazioni.

### Nome della licenza
{: #caveonix_ordering-config-license-name}

Specifica un nome univoco per la licenza.

### Note della licenza
{: #caveonix_ordering-config-license-notes}

Immetti le note per l'istanza del servizio. Puoi modificare le note successivamente nella pagina dei dettagli della licenza.

### Numero di VM per cui acquisire una Licenza
{: #caveonix_ordering-config-vms}

Seleziona il numero di VM per cui deve essere concessa una licenza (da 10 a un massimo di 1000 licenze di VM in incrementi di 10). Per la gestione delle licenze sono richieste almeno 10 VM.

Dopo che hai ordinato il servizio Caveonix RiskForesight on {{site.data.keyword.cloud_notm}}, se desideri una concessione in licenza di ulteriori VM rispetto a quanto hai specificato nell'ordine, puoi sostituire la tua licenza Caveonix RiskForesight ordinando una licenza pi√π grande, copiando la licenza nella tua console Caveonix RiskForesight ed eliminando quindi la licenza precedente dalla console {{site.data.keyword.vmwaresolutions_short}}. Per ulteriori informazioni, vedi [Procedura per ordinare licenze Caveonix RiskForesight](/docs/services/vmwaresolutions?topic=vmware-solutions-caveonix_license_ordering#caveonix_license_ordering-procedure) e [Procedura per eliminare licenze Caveonix RiskForesight](/docs/services/vmwaresolutions?topic=vmware-solutions-caveonix_license_managing#caveonix_license_managing_procedure-delete).

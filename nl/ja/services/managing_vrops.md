# VMware vRealize Operations and vRealize Log Insight on IBM Cloud の管理
{: #managing_vrops}

## vRealize Operations Manager コンソールへのアクセス
{: #managing_vrops-access-vrops-console}

vRealize Operations Manager コンソールにアクセスするには、VMware vRealize Operations and vRealize Log Insight on {{site.data.keyword.cloud_notm}} のサービス詳細ページで**「vRealize Operations Manager Console」**をクリックしてから、同じサービス詳細ページにリストされている資格情報を使用してログインします。

## vSAN のアダプター・インスタンスの開始
{: #managing_vrops-start-adapter}

vSAN の Management Pack アダプター・インスタンスは手動で開始する必要があります。以下のステップを実行します。

1. vRealize Operations Manager コンソールにログインします。
2. vSAN アダプターをクリックします。
3. 「編集」をクリックしてから、接続をテストして、プロンプトが出されたときに証明書を受け入れます。
4. アダプター・インスタンスを開始します。

## vRealize Log Insight コンソールへのアクセス
{: #managing_vrops-access-vlog-console}

vRealize Log Insight コンソールにアクセスするには、VMware vRealize Operations and vRealize Log Insight on {{site.data.keyword.cloud_notm}} のサービス詳細ページで**「vRealize Log Insight Console」**をクリックしてから、同じサービス詳細ページにリストされている資格情報を使用してログインします。

# Commande de VMware vRealize Operations and vRealize Log Insight on IBM Cloud
{: #vrops_ordering}

Vous pouvez commander le service VMware vRealize Operations and vRealize Log Insight on {{site.data.keyword.cloud}} lors de la commande d'une nouvelle instance VMware vCenter Server avec le service inclus ou en ajoutant le service à votre instance existante.

## Commande de VMware vRealize Operations and vRealize Log Insight on IBM Cloud pour une nouvelle instance
{: #vrops_ordering-new}

Vous pouvez commander une nouvelle instance avec VMware vRealize Operations and vRealize Log Insight on {{site.data.keyword.cloud_notm}} en utilisant l'une des méthodes suivantes :
* Depuis la console {{site.data.keyword.vmwaresolutions_short}}, lorsque vous commandez une nouvelle instance, sélectionnez **VMware vRealize Operations and vRealize Log Insight on IBM Cloud** dans la section **Services**.
* Dans le catalogue {{site.data.keyword.cloud_notm}}, cliquez sur l'icône **VMware** dans le panneau de navigation de gauche, puis cliquez sur la carte **VMware vRealize Operations and vRealize Log Insight on IBM Cloud** dans la section **Services VMware**. Spécifiez les paramètres de service et sélectionnez **Ajouter à une nouvelle instance**.

## Commande de vRealize on IBM Cloud pour une instance existante
{: #vrops_ordering-existing}

Vous pouvez ajouter le service vRealize on {{site.data.keyword.cloud_notm}} dans une instance existante en utilisant l'une des méthodes suivantes :
* Depuis la console {{site.data.keyword.vmwaresolutions_short}}, affichez l'instance pour laquelle vous souhaitez ajouter le service, cliquez sur **Services** dans le panneau de navigation de gauche, puis cliquez sur **Ajouter**.
* Dans le catalogue {{site.data.keyword.cloud_notm}}, cliquez sur l'icône **VMware** dans le panneau de navigation de gauche, puis cliquez sur la carte **VMware vRealize Operations and vRealize Log Insight on IBM Cloud** dans la section **Services VMware**. Spécifiez les paramètres de service et sélectionnez **Ajouter à une instance existante**.

## Configuration de vRealize on IBM Cloud
{: #vrops_ordering-config}

Lorsque vous commandez le service VMware vRealize Operations and vRealize Log Insight on {{site.data.keyword.cloud_notm}}, vous devez indiquer si vous voulez louer (inclure avec l'achat) les licences VMware vRealize Operations et VMware vRealize LogInsight d'{{site.data.keyword.cloud_notm}} ou apporter vos propres licences (BYOL).

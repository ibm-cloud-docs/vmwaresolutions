# Commande de HyTrust CloudControl on IBM Cloud

Vous pouvez commander le service HyTrust CloudControl on {{site.data.keyword.cloud_notm}} lors de la commande d'une nouvelle instance avec une paire à haute disponibilité de dispositifs HyTrust CloudControl incluse ou vous pouvez ajouter le dispositif HyTrust CloudControl à votre instance existante.

## Commande de HyTrust CloudControl on IBM Cloud pour une nouvelle instance

Vous pouvez commander une nouvelle instance avec HyTrust CloudControl on {{site.data.keyword.cloud_notm}} à l'aide de l'une des méthodes suivantes :
* Depuis la console {{site.data.keyword.vmwaresolutions_full}}, lorsque vous commandez une nouvelle instance, sélectionnez **HyTrust CloudControl on IBM Cloud** dans la section **Services**.
* Depuis le catalogue {{site.data.keyword.cloud_notm}}, sélectionnez **HyTrust CloudControl on {{site.data.keyword.cloud_notm}}**, spécifiez les paramètres de service et sélectionnez **Ajouter à une nouvelle instance**.

## Commande de HyTrust CloudControl on IBM Cloud pour une instance existante

Vous pouvez ajouter le service HyTrust CloudControl on {{site.data.keyword.cloud_notm}} dans une instance existante à l'aide de l'une des méthodes suivantes :
* Depuis la console {{site.data.keyword.vmwaresolutions_short}}, affichez l'instance pour laquelle vous souhaitez ajouter le service, cliquez sur **Services** dans le panneau de navigation de gauche, puis cliquez sur **Ajouter un service**.
* Depuis le catalogue {{site.data.keyword.cloud_notm}}, sélectionnez **HyTrust CloudControl on IBM Cloud**, spécifiez les paramètres de service et sélectionnez **Ajouter à une instance existante**.

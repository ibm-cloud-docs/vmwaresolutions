# Commande de HyTrust CloudControl on IBM Cloud
{: #htcc_ordering}

Vous pouvez commander le service HyTrust CloudControl on {{site.data.keyword.cloud}} lorsque vous commandez une nouvelle instance avec une paire à haute disponibilité de dispositifs HyTrust DataControl incluse ou en ajoutant les dispositifs HyTrust DataControl à votre instance existante.

## Commande de HyTrust CloudControl on IBM Cloud pour une nouvelle instance
{: #htcc_ordering-new}

Vous pouvez commander une nouvelle instance avec HyTrust CloudControl on {{site.data.keyword.cloud_notm}} en utilisant l'une des méthodes suivantes :
* Depuis la console {{site.data.keyword.vmwaresolutions_short}}, lorsque vous commandez une nouvelle instance, sélectionnez **HyTrust CloudControl on IBM Cloud** dans la section **Services**.
* Depuis le catalogue {{site.data.keyword.cloud_notm}}, sélectionnez **HyTrust CloudControl on IBM Cloud**, spécifiez les paramètres de service et sélectionnez **Ajouter à une nouvelle instance**.

## Commande de HyTrust CloudControl on IBM Cloud pour une instance existante
{: #htcc_ordering-existing}

Vous pouvez ajouter le service HyTrust CloudControl on {{site.data.keyword.cloud_notm}} dans une instance existante en utilisant l'une des méthodes suivantes :
* Depuis la console {{site.data.keyword.vmwaresolutions_short}}, affichez l'instance pour laquelle vous souhaitez ajouter le service, cliquez sur **Services** dans le panneau de navigation de gauche, puis cliquez sur **Ajouter**.
* Depuis le catalogue {{site.data.keyword.cloud_notm}}, sélectionnez **HyTrust CloudControl on IBM Cloud**, spécifiez les paramètres de service et sélectionnez **Ajouter à une instance existante**.

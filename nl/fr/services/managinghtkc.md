# Gestion de HyTrust KeyControl on IBM Cloud

Pour gérer le service HyTrust KeyControl on {{site.data.keyword.cloud}} (HTKC), accédez à l'interface graphique Web HTKC à partir de la console {{site.data.keyword.vmwaresolutions_short}} ou accédez à la console HTKC à partir du client Web vSphere.

## Accès à l'interface graphique Web HyTrust KeyControl à partir de la console IBM Cloud for VMware Solutions

Pour vous connecter à l'interface graphique Web du dispositif HTKC principal ou secondaire, utilisez les données d'identification de l'interface graphique Web qui figurent sur la page des détails du service HyTrust KeyControl on {{site.data.keyword.cloud_notm}}.

## Accès à la console HyTrust KeyControl à partir du client Web vSphere

Pour accéder à la console HTKC à partir du client Web vSphere, procédez comme suit :
1. Dans le client Web vSphere, localisez les machines virtuelles nommées **KC1** et **KC2** dont l'adresse IP correspondante figure sur la page des détails du service HyTrust KeyControl on {{site.data.keyword.cloud_notm}}.
2. Cliquez avec le bouton droit de la souris sur **KC1** ou **KC2**, puis cliquez sur **Ouvrir la console**.
3. Connectez-vous à la console à l'aide des données d'identification de console qui figurent sur la page des détails du service HyTrust KeyControl on {{site.data.keyword.cloud_notm}}.

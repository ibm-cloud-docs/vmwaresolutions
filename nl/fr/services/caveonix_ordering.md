# Commande de Caveonix RiskForesight on IBM Cloud
{: #caveonix_ordering}

Vous pouvez commander le service Caveonix RiskForesight on {{site.data.keyword.cloud}} lors de la commande d'une nouvelle instance VMware vCenter Server avec le service déjà inclus ou vous pouvez ajouter ce service à votre instance existante.

## Commande de Caveonix RiskForesight on IBM Cloud pour une nouvelle instance
{: #caveonix_ordering-new}

Pour commander une nouvelle instance VMware vCenter Server on {{site.data.keyword.cloud_notm}} with Caveonix RiskForesight on {{site.data.keyword.cloud_notm}}, sélectionnez **Caveonix RiskForesight on IBM Cloud** dans la section **Services** lorsque vous commandez l'instance depuis la console {{site.data.keyword.vmwaresolutions_short}}.

## Commande de Caveonix RiskForesight on IBM Cloud pour une instance existante
{: #caveonix_ordering-existing}

Pour ajouter le service Caveonix RiskForesight on {{site.data.keyword.cloud_notm}} dans une instance VMware vCenter Server on {{site.data.keyword.cloud_notm}} existante, affichez l'instance pour laquelle vous souhaitez ajouter le service, cliquez sur **Services** dans le panneau de navigation de gauche, puis cliquez sur **Ajouter**.

## Configuration de Caveonix RiskForesight on IBM Cloud
{: #caveonix_ordering-config}

Lorsque vous commandez le service Caveonix RiskForesight on {{site.data.keyword.cloud_notm}}, fournissez les paramètres suivants :

### Nom de licence
{: #caveonix_ordering-config-license-name}

Indiquez un nom unique pour la licence.

### Remarques sur la licence
{: #caveonix_ordering-config-license-notes}

Indiquez les remarques relatives à l'instance de service. Vous pourrez éditer les remarques ultérieurement sur la page des détails de licence.

### Nombre de machines virtuelles pour lesquelles concéder une licence
{: #caveonix_ordering-config-vms}

Sélectionnez le nombre de licences pour lesquelles concéder une licence (10 à 1000 licences de machine virtuelle au maximum, par incréments de 10). Un minimum de 10 machines virtuelles est requis pour la gestion des licences.

Après avoir commandé le service Caveonix RiskForesight on {{site.data.keyword.cloud_notm}}, si vous souhaitez obtenir plus de machines virtuelles sous licence que ce qui est spécifié dans la commande, vous pouvez remplacer votre licence Caveonix RiskForesight en commandant une licence plus importante, en copiant la licence sur votre console Caveonix RiskForesight, pusi en supprimant la licence précédente de la console {{site.data.keyword.vmwaresolutions_short}}. Pour plus d'informations, voir [Procédure de commande de licences Caveonix RiskForesight](/docs/services/vmwaresolutions?topic=vmware-solutions-caveonix_license_ordering#caveonix_license_ordering-procedure) et [Procédure de suppression de licences Caveonix RiskForesight](/docs/services/vmwaresolutions?topic=vmware-solutions-caveonix_license_managing#caveonix_license_managing_procedure-delete).

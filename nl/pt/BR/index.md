# Introdução ao IBM Cloud for VMware Solutions
{: #gettingstarted}

O console do {{site.data.keyword.vmwaresolutions_full}} é a interface em que você pede e gerencia suas implementações. Cada implementação é gerenciada como uma instância no console. O console é uma interface com o usuário independente separada do {{site.data.keyword.slportal}}.

## Antes de iniciar

Revise informações importantes sobre os requisitos do navegador e as contas do usuário antes de começar a trabalhar com o {{site.data.keyword.vmwaresolutions_short}}.

### Requisitos de navegador e de resolução

Os seguintes navegadores são suportados:
*  Mozilla Firefox
*  Google Chrome
*  Apple Safari

**Nota**: o Microsoft Internet Explorer não é suportado.

Para obter uma visualização ideal e trabalhar no console do {{site.data.keyword.vmwaresolutions_short}}, configure a resolução da tela para pelo menos 1024 px de largura por 500 px de altura.

### IBMid

O **IBMid** permite usar um nome de usuário de login para todos os produtos e serviços IBM que você usa, incluindo o {{site.data.keyword.cloud_notm}}. O {{site.data.keyword.vmwaresolutions_short}} é fornecido como uma solução de infraestrutura no catálogo do {{site.data.keyword.cloud_notm}}. Portanto, para acessar o console do {{site.data.keyword.vmwaresolutions_short}}, deve-se ter um **IBMid**.

Para usar seu **IBMid** para efetuar login no console do {{site.data.keyword.vmwaresolutions_short}}, deve-se associar o **IBMid** a uma conta do {{site.data.keyword.cloud_notm}}. Ao efetuar login no console pela primeira vez, você é orientado a associar seu **IBMid** existente a uma conta do {{site.data.keyword.cloud_notm}} ou a se inscrever para uma nova conta do {{site.data.keyword.cloud_notm}} que é associada automaticamente a seu **IBMid**. É necessário passar por esse processo apenas uma vez.

Se você tiver problemas ao associar seu **IBMid** a uma conta {{site.data.keyword.cloud_notm}}, veja [Resolução de problemas para acessar o {{site.data.keyword.cloud_notm}}](https://console.bluemix.net/docs/troubleshoot/ts_accessing.html).

### Conta do IBM Cloud

A conta do {{site.data.keyword.cloud_notm}} é necessária para pedir e usar serviços do IBM Cloud. As informações de faturamento são associadas à conta do IBM Cloud. Portanto, para pedir e usar as ofertas de implementação do {{site.data.keyword.vmwaresolutions_short}}, deve-se ter uma conta do {{site.data.keyword.cloud_notm}}. O custo da infraestrutura física e virtual e as licenças resultantes são debitados de sua conta do {{site.data.keyword.cloud_notm}}. Para obter mais informações sobre os requisitos que sua conta do {{site.data.keyword.cloud_notm}} deve atender, veja os requisitos de conta do [{{site.data.keyword.cloud_notm}}](vmonic/slaccountrequirement.html).

### Conta do IBM Cloud Infrastructure (SoftLayer)

A conta de infraestrutura do {{site.data.keyword.cloud_notm}} (SoftLayer) era conhecida anteriormente como a conta do IBM SoftLayer.

É possível vincular contas de infraestrutura do {{site.data.keyword.cloud_notm}} (SoftLayer) a contas do {{site.data.keyword.cloud_notm}} para usar recursos combinados de infraestrutura como serviço (IaaS) e plataforma como serviço (PaaS). Em seguida, é possível acessar recursos de IaaS e
de PaaS por meio de um login único. Vincular suas contas também fornece uma única fatura para todos os recursos PaaS e IaaS que você usa.

* Se você não tiver uma conta de infraestrutura do {{site.data.keyword.cloud_notm}} (SoftLayer), solicite uma seguindo o procedimento em [Inscrevendo-se para uma conta de infraestrutura do IBM Cloud (SoftLayer)](../vmonic/signing_softlayer_account.html) e, em seguida, vincule sua conta de infraestrutura do {{site.data.keyword.cloud_notm}} (SoftLayer) à sua conta do {{site.data.keyword.cloud_notm}} seguindo o procedimento em [Vinculando contas do IBMid](https://console.bluemix.net/docs/account/softlayerlink.html).
* Se você já tiver uma conta de infraestrutura do {{site.data.keyword.cloud_notm}} (SoftLayer), será possível vinculá-la à sua conta do {{site.data.keyword.cloud_notm}} seguindo o procedimento em [Vinculando contas do IBMid](https://console.bluemix.net/docs/account/softlayerlink.html).

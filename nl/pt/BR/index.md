# Introdução ao IBM Cloud for VMware Solutions
{: #gettingstarted}

O console do {{site.data.keyword.vmwaresolutions_full}} é a interface em que você pede e gerencia suas implementações. Cada implementação é gerenciada como uma instância no console. O console é uma interface com o usuário independente separada do {{site.data.keyword.slportal_full}}.

## Antes de iniciar

Revise informações importantes sobre os requisitos do navegador e as contas do usuário antes de começar a trabalhar com o {{site.data.keyword.vmwaresolutions_short}}.

### Requisitos de navegador e de resolução

Os seguintes navegadores são suportados:
*  Mozilla Firefox
*  Google Chrome
*  Apple Safari

**Nota**: o Microsoft Internet Explorer não é suportado.

Para obter uma visualização ideal e trabalhar no console do {{site.data.keyword.vmwaresolutions_short}}, configure a resolução da tela para pelo menos 1024 px de largura por 500 px de altura.

### Conta IBMid

Para acessar o console do {{site.data.keyword.vmwaresolutions_short}}, deve-se ter uma conta do **IBMid**.

O {{site.data.keyword.vmwaresolutions_short}} é fornecido como uma solução de infraestrutura no catálogo do {{site.data.keyword.cloud_notm}}. Portanto, para usar sua conta do **IBMid** para efetuar login no console do {{site.data.keyword.vmwaresolutions_short}}, deve-se associar a conta do **IBMid** a uma conta do {{site.data.keyword.cloud_notm}}.

Ao efetuar login no console pela primeira vez, você é orientado a associar sua conta do **IBMid** a uma conta do {{site.data.keyword.cloud_notm}} ou a se inscrever em uma nova conta do {{site.data.keyword.cloud_notm}} que é automaticamente associada à sua conta do **IBMid**. É necessário passar por esse processo apenas uma vez.

Se tiver problemas ao associar sua conta do **IBMid** a uma conta do {{site.data.keyword.cloud_notm}}, veja [Resolução de problemas para acessar o {{site.data.keyword.cloud_notm}}](https://console.bluemix.net/docs/troubleshoot/ts_accessing.html){:new_window}.

### Conta do IBM Cloud

Para usar as ofertas de implementação, deve-se ter uma conta do {{site.data.keyword.cloud_notm}}. O custo da infraestrutura física e virtual e as licenças resultantes são debitados de sua conta do {{site.data.keyword.cloud_notm}}. Para obter mais informações sobre os requisitos que sua conta do {{site.data.keyword.cloud_notm}} deve atender, veja os requisitos de conta do [{{site.data.keyword.cloud_notm}}](vmonic/slaccountrequirement.html).

**Nota**: a conta {{site.data.keyword.cloud_notm}} era conhecida anteriormente como a conta IBM SoftLayer.

É possível associar apenas uma conta do {{site.data.keyword.cloud_notm}} a uma conta do {{site.data.keyword.vmwaresolutions_short}}.

## Acessando o console do IBM Cloud for VMware Solutions

Para acessar o console por meio do catálogo do {{site.data.keyword.cloud_notm}}, veja [Acessando o console do {{site.data.keyword.vmwaresolutions_short}}](vmonic/loginmethod.html).

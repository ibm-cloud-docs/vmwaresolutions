# 訂購 VMware vRealize Operations 及 vRealize Log Insight on IBM Cloud
{: #vrops_ordering}

您可以在訂購包含 VMware vRealize Operations 及 vRealize Log Insight on {{site.data.keyword.cloud}} 服務的新 VMware vCenter Server 服務時訂購該新服務，也可以將該服務新增至現有實例。

## 為新實例訂購 VMware vRealize Operations 及 vRealize Log Insight on IBM Cloud
{: #vrops_ordering-new}

您可以使用下列其中一種方法，訂購包含 VMware vRealize Operations 及 vRealize Log Insight on {{site.data.keyword.cloud_notm}} 的新實例：
* 當您從 {{site.data.keyword.vmwaresolutions_short}} 主控台訂購新實例時，請選取**服務**區段中的 **VMware vRealize Operations 及 vRealize Log Insight on IBM Cloud**。
* 從 {{site.data.keyword.cloud_notm}} 型錄中，按一下左導覽窗格上的 **VMware** 圖示，然後按一下 **VMware Services** 區段中的 **VMware vRealize Operations 及 vRealize Log Insight on IBM Cloud** 卡。指定服務設定，然後選取**新增至新實例**。

## 為現有實例訂購 vRealize on IBM Cloud
{: #vrops_ordering-existing}

您可以使用下列其中一種方法，將 vRealize on {{site.data.keyword.cloud_notm}} 服務新增至現有實例：
* 從 {{site.data.keyword.vmwaresolutions_short}} 主控台，檢視您要為其新增服務的實例，按一下左導覽窗格上的**服務**，然後按一下**新增**。
* 從 {{site.data.keyword.cloud_notm}} 型錄中，按一下左導覽窗格上的 **VMware** 圖示，然後按一下 **VMware Services** 區段中的 **VMware vRealize Operations 及 vRealize Log Insight on IBM Cloud** 卡。指定服務設定，然後選取**新增至現有實例**。

## vRealize on IBM Cloud 配置
{: #vrops_ordering-config}

訂購 VMware vRealize Operations 及 vRealize Log Insight on {{site.data.keyword.cloud_notm}} 服務時，必須指定是要向 {{site.data.keyword.cloud_notm}} 租用（購買隨附）VMware vRealize Operations 和 VMware vRealize Log Insight 授權，還是自帶授權 (BYOL)。

# 管理 FortiGate Virtual Appliance on IBM Cloud

## 存取 FortiGate 主控台

若要管理 FortiGate Virtual Appliance on {{site.data.keyword.cloud}} 服務，您必須使用下列其中一種方式來存取 FortiGate® 主控台：
* 使用您可以在 FortiGate Virtual Appliance on {{site.data.keyword.cloud_notm}} 服務詳細資料頁面上找到的認證，登入 FortiOS Web Client。
* 使用您可以在 FortiGate Virtual Appliance on {{site.data.keyword.cloud_notm}} 服務詳細資料頁面上找到的認證，透過 SSH 連線存取主控台。

如需相關資訊，請參閱下列主題：
* [訂購、檢視及移除 Cloud Foundation 實例的服務](../sddc/sd_addingremovingservices.html)
* [訂購、檢視及移除 vCenter Server 實例的服務](../vcenter/vc_addingremovingservices.html)

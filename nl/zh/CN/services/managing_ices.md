# 请求 IBM Cloud Expert Services
{: #managing_ices}

{{site.data.keyword.cloud}} Expert Services 可以帮助您采用 VMware on IBM Cloud 来实现业务现代化并将业务部署在灵活、安全的平台上。利用 Expert Services，可以加速实现价值。

{{site.data.keyword.cloud_notm}} Expert Services 是一组服务产品，旨在引导您部署、迁移和维护您自己的 {{site.data.keyword.cloud_notm}} 解决方案，可以是从规划到现代化的所有阶段，或中间的任何一个阶段。Expert Services 团队会分五个阶段来复查您的基础架构需求：发现、了解、规划、构建和运营。

您可以从包含此方法的一系列产品中进行选择，以设计 VMware 环境并使您的业务焕然一新。

## IBM Cloud Expert Services 的技术规范
{: #managing_ices-specs}

预定义的 Expert Services 包含 OnDemand Consulting、Jumpstart for {{site.data.keyword.cloud_notm}} vCenter Server、Jumpstart for VMware Hybridity Bundle，以及 Architectural Consultancy 研讨会。

Expert Services OnDemand Consulting (ODC) 针对有技术员工的客户，他们关注的是当他们遇到问题时能获得及时、灵活、专享的技术支援，以及在与 IBM 积极沟通的同时获得额外指导。ODC 在客户需要时通过单点联系人为其提供引领，提供解决方案建议，从而推动实现基本业务成功。

Hybridity Bundle Jumpstart 简化了将 VMware 工作负载从内部部署或其他云提供者移动到 IBM Cloud 的过程。Expert Services 将引导您完成初始评估，概要介绍 IBM Cloud for VMware Solutions with HCX，以及设计和规划。其中包括前 20 个 VM 的基础架构订购、配置和迁移的伴随指导。

此产品通过评估客户的虚拟化需求，概要介绍 IBM Cloud for VMware Solutions，然后设计和规划其 VMWare 解决方案，从而加快采用 VMware on IBM Cloud 的速度。然后 IBM 会提供有关基础架构订购和初始配置的伴随指导。后续还可以选择 OnDemand Consulting 来提供持续的客户协助。

### 标准功能
{: #managing_ices-standard-feat}

* 咨询专家
* 技术技能支持
* 主题专家 (SME) 指导时间：每月 1 小时远程 SME 时间
* 专用联系点

### 高级功能
{: #managing_ices-premium-feat}

* 每周联锁会议
* SME 指导时间：每月两天的 SME 时间

## IBM Cloud Expert Services 的主要优点
{: #managing_ices-benefits}

* 咨询参与方法以您的业务目标为中心
* 属于 IBM Cloud Platform 团队的经验丰富的体系结构专业知识
* 通过经过验证的成功实施路径来加快云采用速度
* 使用 IBM Cloud 产品和功能来实施解决方案的规范性最佳实践
* 在 IBM Cloud 专家的帮助下，快速学习并降低风险

## 请求 IBM Cloud Expert Services 的受管服务的过程
{: #managing_ices-procedure}

1. 在 {{site.data.keyword.vmwaresolutions_full}} 控制台中，单击左侧导航窗格中的**开始**。
2. 向下滚动该页面，在**订购更多受管服务**下，单击 **IBM Cloud Expert Services 的受管服务**卡。
3. 在 {{site.data.keyword.cloud_notm}} Expert Services 页面的**参与 Cloud Expert Services 预售**框中，单击**请求咨询**。

  {{site.data.keyword.vmwaresolutions_short}} 代表将通过您的 {{site.data.keyword.cloud_notm}} 联系人信息与您联系，以帮助您获取需要的解决方案。

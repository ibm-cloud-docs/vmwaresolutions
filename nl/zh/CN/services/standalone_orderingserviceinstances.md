# 订购内部部署 VMware HCX on IBM Cloud 实例
{: #standalone_orderingserviceinstances}

可以订购内部部署 HCX on {{site.data.keyword.cloud}} 实例，而不将其与任何 vCenter Server with Hybridity Bundle 实例关联，以对内部部署 HCX on {{site.data.keyword.cloud_notm}} 安装进行许可和激活。

## 开始之前
{: #standalone_orderingserviceinstances-reqs}

确保已完成以下任务：
*  已在**设置**页面上配置 {{site.data.keyword.cloud_notm}} 基础架构凭证。有关更多信息，请参阅[用户帐户和设置](/docs/services/vmwaresolutions/vmonic?topic=vmware-solutions-useraccount)。
*  已查看[安装内部部署 HCX on {{site.data.keyword.cloud_notm}} 实例时的注意事项](/docs/services/vmwaresolutions/services?topic=vmware-solutions-standalone_considerations)中的所有注意事项。

## 订购内部部署 VMware HCX on IBM Cloud 实例的过程
{: #standalone_orderingserviceinstances-procedure}

1. 在 {{site.data.keyword.vmwaresolutions_short}} 控制台中，单击左侧导航窗格上的**资源**。
2. 滚动到**内部部署 HCX 实例**表，然后单击**供应新项**。
3. 指定**内部部署实例名称**。
4. 单击订单适用条款的链接，并确保在订购实例之前同意这些条款。
5. 单击**供应**。

## 结果
{: #standalone_orderingserviceinstances-results}

服务激活密钥的订购会自动启动。您将收到说明订单正在处理的确认，并且您可以通过查看实例详细信息来检查订单的状态。

激活密钥准备就绪可供使用后，该实例的状态会更改为**已安装**，并且您将收到通过电子邮件发送的通知。

# 请求 Veeam on IBM Cloud 的受管服务

Veeam on {{site.data.keyword.cloud}} 服务可与 VMware 系统管理程序无缝集成，以帮助企业实现高可用性 (HA)。此服务可为应用程序和数据提供恢复点和时间目标。在完成配置后的 15 分钟内就会提供恢复点和时间目标。使用此服务可在 Veeam 控制台中直接控制基础架构的所有虚拟机的备份和复原。

IBM Resiliency Backup as a Service 提供各种稳健的现场、非现场和基于混合云的数据保护解决方案，支持实现信息灾备能力。有关更多信息，请参阅 [IBM Resiliency Backup as a Service](https://www.ibm.com/us-en/marketplace/managed-backup-services)。

请求 Veeam on {{site.data.keyword.cloud_notm}} 的受管服务后，请使用 Veeam 备份软件和 IBM Resiliency Backup as a Service 来部署完全受管的备份环境。此组合可通过以下方式为您提供帮助：
* 您有多个数据保护解决方案可供选择，以支持对专用内部部署环境、{{site.data.keyword.cloud_notm}} 基础架构或 IBM Resiliency Services Center（作为主位置或辅助位置）中的关键业务数据进行高度安全的自动化保护。
* 还可获取可扩展的灵活数据保护解决方案，以减少总体拥有成本，并管理合规性需求。

## 请求 Veeam on IBM Cloud 的受管服务的过程

1. 在 {{site.data.keyword.vmwaresolutions_short}} 控制台中，单击左侧导航窗格中的**开始**。
2. 向下滚动该页面，在**订购更多受管服务**下，单击 **Veeam on IBM Cloud 的受管服务**卡。
3. 在 **Veeam on IBM Cloud** 页面上，查看 Veeam on {{site.data.keyword.cloud_notm}} 作为受管服务的描述和技术规范，然后单击**创建**。
4. 根据需求指定配置设置，或者接受缺省值。
5. 单击 **vCenter Server** 或 **Cloud Foundation** 以将相应服务添加到其中一个实例。
6. 要在订购新实例期间添加服务，请单击**添加到新实例**，然后继续订购新的 [vCenter Server](/docs/services/vmwaresolutions/vcenter/vc_orderinginstance.html)、[vCenter Server with Hybridity](/docs/services/vmwaresolutions/vcenter/vc_hybrid_orderinginstance.html) 或 [Cloud Foundation](/docs/services/vmwaresolutions/sddc/sd_orderinginstance.html) 实例。
7. 要将服务添加到现有实例，请单击**添加到现有实例**，从列表中选择所需实例，然后通过单击**供应**以确认要继续处理订单。

# 管理 FortiGate Virtual Appliance on IBM Cloud
{: #managingfortinetvm}

## 访问 FortiGate 控制台
{: #managingfortinetvm-access-console}

要管理 FortiGate Virtual Appliance on {{site.data.keyword.cloud}} 服务，必须通过下列其中一种方式来访问 FortiGate® 控制台：
* 使用可在 FortiGate Virtual Appliance on {{site.data.keyword.cloud_notm}} 服务详细信息页面上找到的凭证，登录到 FortiOS Web Client。
* 使用可在 FortiGate Virtual Appliance on {{site.data.keyword.cloud_notm}} 服务详细信息页面上找到的凭证，通过 SSH 连接访问该控制台。

有关更多信息，请参阅以下主题：
* [订购、查看和除去 Cloud Foundation 实例的服务](/docs/services/vmwaresolutions/sddc?topic=vmware-solutions-sd_addingremovingservices)
* [订购、查看和除去 vCenter Server 实例的服务](/docs/services/vmwaresolutions/vcenter?topic=vmware-solutions-vc_addingremovingservices)

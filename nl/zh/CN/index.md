# IBM Cloud for VMware Solutions 入门
{: #gettingstarted}

{{site.data.keyword.vmwaresolutions_full}} 控制台是用于订购和管理部署的界面。每个部署在控制台中作为一个实例进行管理。控制台是与 {{site.data.keyword.slportal}} 分开的独立用户界面。

## 开始之前

开始使用 {{site.data.keyword.vmwaresolutions_short}} 之前，请查看有关浏览器需求和用户帐户的重要信息。

### 浏览器和分辨率需求

支持以下浏览器：
*  Mozilla Firefox
*  Google Chrome
*  Apple Safari

**注**：不支持 Microsoft Internet Explorer。

为了使 {{site.data.keyword.vmwaresolutions_short}} 控制台上的查看和工作达到最佳效果，请将屏幕分辨率设置为至少 1024 x 500 像素（宽 x 高）。

### IBM 标识

通过 **IBM 标识**，您可以对使用的所有 IBM 产品和服务（包括 {{site.data.keyword.cloud_notm}}）使用一个登录用户名。{{site.data.keyword.vmwaresolutions_short}} 在 {{site.data.keyword.cloud_notm}}“目录”中作为基础架构解决方案提供。因此，要访问 {{site.data.keyword.vmwaresolutions_short}} 控制台，您必须具有 **IBM 标识**。

要使用 **IBM 标识**登录到 {{site.data.keyword.vmwaresolutions_short}} 控制台，必须将 **IBM 标识**与 {{site.data.keyword.cloud_notm}} 帐户相关联。首次登录到控制台时，系统会引导您将现有 **IBM 标识**与 {{site.data.keyword.cloud_notm}} 帐户相关联，或者引导您注册与 **IBM 标识**自动关联的新 {{site.data.keyword.cloud_notm}} 帐户。您只需要完成此过程一次。

如果将 **IBM 标识**与 {{site.data.keyword.cloud_notm}} 帐户相关联时发生问题，请参阅 [{{site.data.keyword.cloud_notm}} 访问故障诊断](https://console.bluemix.net/docs/troubleshoot/ts_accessing.html)。

### IBM Cloud 帐户

订购并使用 IBM Cloud 服务需要 {{site.data.keyword.cloud_notm}} 帐户。计费信息与 IBM Cloud 帐户相关联。因此，要订购并使用 {{site.data.keyword.vmwaresolutions_short}} 的部署产品，您必须具有 {{site.data.keyword.cloud_notm}} 帐户。物理和虚拟基础架构的成本以及生成的许可证都将向您的 {{site.data.keyword.cloud_notm}} 帐户收费。有关 {{site.data.keyword.cloud_notm}} 帐户必须满足的需求的更多信息，请参阅 [{{site.data.keyword.cloud_notm}} 帐户需求](vmonic/slaccountrequirement.html)。

### IBM Cloud Infrastructure (SoftLayer) 帐户

{{site.data.keyword.cloud_notm}} Infrastructure (SoftLayer) 帐户先前称为 IBM SoftLayer 帐户。

您可以将 {{site.data.keyword.cloud_notm}} Infrastructure (SoftLayer) 帐户与 {{site.data.keyword.cloud_notm}} 帐户相链接，以使用组合基础架构即服务 (IaaS) 和平台即服务 (PaaS) 资源。然后，可以通过一次登录来访问 IaaS 资源和 PaaS 资源。通过链接帐户，您使用的所有 PaaS 和 IaaS 资源的费用将包含在一张发票中。

* 如果您没有 {{site.data.keyword.cloud_notm}} Infrastructure (SoftLayer) 帐户，请遵循[注册 IBM Cloud Infrastructure (SoftLayer) 帐户](../vmonic/signing_softlayer_account.html)中的过程请求 Infrastructure (SoftLayer) 帐户，然后遵循[链接 IBM 标识帐户](https://console.bluemix.net/docs/account/softlayerlink.html)中的指示信息将 {{site.data.keyword.cloud_notm}} Infrastructure (SoftLayer) 帐户与 {{site.data.keyword.cloud_notm}} 帐户相链接。
* 如果您已具有 {{site.data.keyword.cloud_notm}} Infrastructure (SoftLayer) 帐户，那么可以通过遵循[链接 IBM 标识帐户](https://console.bluemix.net/docs/account/softlayerlink.html)中的过程将其与 {{site.data.keyword.cloud_notm}} 帐户相链接。

## 访问 IBM Cloud for VMware Solutions 控制台

要在 {{site.data.keyword.cloud_notm}}“目录”中访问控制台，请参阅[访问 {{site.data.keyword.vmwaresolutions_short}} 控制台](vmonic/loginmethod.html)。

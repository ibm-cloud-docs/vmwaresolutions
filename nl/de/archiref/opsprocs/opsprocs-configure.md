# Konfigurationstasks
{: #opsprocs-configure}

Nachdem Ihre vCenter Server-Instanz bereitgestellt wurde, gibt es eine Reihe von Konfigurationstasks, die von Systemadministratoren ausgeführt werden müssen, um die Startumgebung an die aktuellen Anforderungen ihres Unternehmens anzupassen und für zukünftige Serviceanforderungen vorzubereiten.

Diese Konfigurationstasks sind:

* Instanz- und Clusteranweisungen
* VM-Prozeduren
* vCenter-Prozeduren
* vSphere ESXi-Hostprozeduren
* Speicherprozeduren
* Netzprozeduren

Sie können die vorhandene Namenskonvention Ihres Unternehmens erweitern oder eine neue erstellen. Weitere Informationen finden Sie unter [Namenskonvention in VMware Validated Design 4.1](https://docs.vmware.com/en/VMware-Validated-Design/4.1/com.vmware.vvd.sddc-upgrade.doc/GUID-FCA126F1-E429-454E-B3BA-226525742DC1.html){:new_window}. | Konfiguration der Namensauflösung |  Um diese Funktionen in vCenter zu nutzen, fügen Sie eine Namensauflösung für die vCenter Server-Komponenten (abhängig von Ihrer Verbindungsmethode) auf Ihrer Workstation oder Jumpbox hinzu. Weitere Informationen finden Sie unter [OVF-Datei mit VMware vSphere Web Client bereitstellen](/docs/services/vmwaresolutions?topic=vmware-solutions-trbl_deploy_ovf#trbl_deploy_ovf){:new_window}. |

| Konfiguration der vSphere-Überwachung  | vSphere enthält ein benutzerkonfigurierbares Ereignis- und Alarm-Subsystem, das Ereignisse von vSphere-Komponenten verfolgt und die Daten in Protokolldateien und in der vCenter-Datenbank speichert. Ereignisse sind Datensätze von Benutzer- oder Systemaktionen, die für vSphere-Objekte auftreten. Es gibt drei Arten von Ereignissen: Information, Warnung und Fehler. Alarme sind Benachrichtigungen, die als Reaktion auf ein Ereignis, auf einen Satz von Bedingungen oder auf den Status eines vSphere-Objekts aktiviert werden. Für Alarme gibt es die folgenden Prioritätsstufen: Normal – grün, Warnung – gelb und Alert – rot. Alarmaktionen sind Operationen, die als Antwort auf einen Alarm oder eine gesendete E-Mail-Benachrichtigung auftreten. Informationen zur Konfiguration und zur vSphere-Überwachung finden Sie unter [Überwachen von Ereignissen, Alarmen und automatisierten Aktionen](https://docs.vmware.com/en/VMware-vSphere/6.5/com.vmware.vsphere.monitoring.doc/GUID-9272E3B2-6A7F-427B-994C-B15FF8CADC25.html){:new_window}. |

# VMware vRealize Operations und vRealize Log Insight on IBM Cloud - Bestellverfahren
{: #vrops_ordering}

Sie können den Service für VMware vRealize Operations und vRealize Log Insight on {{site.data.keyword.cloud}} bestellen, wenn Sie eine neue VMware vCenter Server-Instanz mit enthaltenem Service bestellen oder den Service zu Ihrer vorhandenen Instanz hinzufügen.

## VMware vRealize Operations und vRealize Log Insight on IBM Cloud für eine neue Instanz - Bestellverfahren
{: #vrops_ordering-new}

Sie können mithilfe einer der folgenden Methoden eine neue Instanz mit VMware vRealize Operations und vRealize Log Insight on {{site.data.keyword.cloud_notm}} bestellen:
* Wählen Sie beim Bestellen einer neuen Instanz über die {{site.data.keyword.vmwaresolutions_short}}-Konsole im Abschnitt **Services** die Option **VMware vRealize Operations und vRealize Log Insight on IBM Cloud** aus.
* Klicken Sie im {{site.data.keyword.cloud_notm}}-Katalog im linken Navigationsfenster auf das Symbol **VMware** und anschließend im Abschnitt **VMware Services** auf die Karte **VMware vRealize Operations und vRealize Log Insight on IBM Cloud**. Geben Sie die Serviceeinstellungen an und wählen Sie **Zu neuer Instanz hinzufügen** aus.

## vRealize on IBM Cloud für eine vorhandene Instanz bestellen
{: #vrops_ordering-existing}

Sie können den Service vRealize on {{site.data.keyword.cloud_notm}} mit einer der folgenden Methoden zu einer vorhandenen Instanz hinzufügen:
* Zeigen Sie über die {{site.data.keyword.vmwaresolutions_short}}-Konsole die Instanz an, für die der Service hinzugefügt werden soll, klicken Sie im linken Navigationsfenster auf **Services** und anschließend auf **Hinzufügen**.
* Klicken Sie im {{site.data.keyword.cloud_notm}}-Katalog im linken Navigationsfenster auf das Symbol **VMware** und anschließend im Abschnitt **VMware Services** auf die Karte **VMware vRealize Operations und vRealize Log Insight on IBM Cloud**. Geben Sie die Serviceeinstellungen an und wählen Sie **Zu vorhandener Instanz hinzufügen** aus.

## vRealize on IBM Cloud - Konfiguration
{: #vrops_ordering-config}

Wenn Sie den Service für VMware vRealize Operations und vRealize Log Insight on {{site.data.keyword.cloud_notm}} bestellen, müssen Sie angeben, ob Sie die VMware vRealize Operations- und VMware vRealize LogInsight-Lizenzen von {{site.data.keyword.cloud_notm}} mieten (in den Kauf einbeziehen) oder Ihre eigene Lizenz nutzen möchten (BYOL).

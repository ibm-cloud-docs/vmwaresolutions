# Iniciación IBM Cloud for VMware Solutions
{: #gettingstarted}

La consola de {{site.data.keyword.vmwaresolutions_full}} es la interfaz con la que solicita y gestiona sus despliegues. Cada despliegue se gestiona como una instancia en la consola. La consola es una interfaz de usuario autónoma, independiente del {{site.data.keyword.slportal_full}}.

## Antes de empezar

Revise la información importante sobre los requisitos del navegador y las cuentas de usuario antes de empezar a trabajar con {{site.data.keyword.vmwaresolutions_short}}.

### Requisitos de navegador y de resolución

Se admiten los siguientes navegadores:
*  Mozilla Firefox
*  Google Chrome
*  Apple Safari

**Nota**: no se admite Microsoft Internet Explorer.

Para ver y trabajar mejor con la consola de {{site.data.keyword.vmwaresolutions_short}}, defina una resolución de pantalla de al menos 1024 px de anchura por 500 px de altura.

### Cuenta de IBMid

Para acceder a la consola de {{site.data.keyword.vmwaresolutions_short}}, debe tener una cuenta de **IBMid**.

{{site.data.keyword.vmwaresolutions_short}} se proporciona como una solución de infraestructura en el catálogo de {{site.data.keyword.cloud_notm}}. Por lo tanto, para utilizar la cuenta de **IBMid** para iniciar una sesión en la consola de {{site.data.keyword.vmwaresolutions_short}}, debe asociar la cuenta de **IBMid** con una cuenta de {{site.data.keyword.cloud_notm}}.

Cuando inicie una sesión en la consola por primera vez, se le indicará que asocie su cuenta de **IBMid** con una cuenta de {{site.data.keyword.cloud_notm}} o que se registre para conseguir una nueva cuenta de {{site.data.keyword.cloud_notm}} que se asociará automáticamente con su cuenta de **IBMid**. Debe pasar por este proceso una sola vez.

Si tiene problemas para asociar su cuenta de **IBMid** con una cuenta de {{site.data.keyword.cloud_notm}}, consulte [Resolución de problemas para acceder a {{site.data.keyword.cloud_notm}}](https://console.bluemix.net/docs/troubleshoot/ts_accessing.html){:new_window}.

### Cuenta de IBM Cloud

Para utilizar las ofertas de despliegue, debe tener una cuenta de {{site.data.keyword.cloud_notm}}. El coste de la infraestructura física y virtual y las licencias resultantes se cargarán a su cuenta de {{site.data.keyword.cloud_notm}}. Para obtener más información sobre los requisitos que debe cumplir su cuenta de {{site.data.keyword.cloud_notm}}, consulte [Requisitos de la cuenta de {{site.data.keyword.cloud_notm}}](vmonic/slaccountrequirement.html).

**Nota**: la cuenta de {{site.data.keyword.cloud_notm}} se conocía anteriormente como cuenta de IBM SoftLayer.

Solo puede asociar una cuenta de {{site.data.keyword.cloud_notm}} a una cuenta de {{site.data.keyword.vmwaresolutions_short}}.

## Acceso a la consola de IBM Cloud for VMware Solutions

Para acceder a la consola desde el catálogo de {{site.data.keyword.cloud_notm}}, consulte [Acceso a la consola de {{site.data.keyword.vmwaresolutions_short}}](vmonic/loginmethod.html).

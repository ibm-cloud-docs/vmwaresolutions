# Solicitud de HyTrust KeyControl on IBM Cloud

Puede solicitar el servicio de HyTrust KeyControl on {{site.data.keyword.cloud}} cuando solicite una nueva instancia con un par de HA de dispositivos de HyTrust KeyControl incluidos o añadiendo los dispositivos de HyTrust KeyControl a la instancia existente.

## Solicitud de HyTrust KeyControl on IBM Cloud para una nueva instancia

Puede solicitar una nueva instancia con HyTrust KeyControl on {{site.data.keyword.cloud_notm}} utilizando uno de los métodos siguientes:
* Desde la consola de {{site.data.keyword.vmwaresolutions_short}}, al solicitar una nueva instancia, seleccione **HyTrust KeyControl on IBM Cloud** en la sección **Servicios**.
* Desde el catálogo de {{site.data.keyword.cloud_notm}}, seleccione **HyTrust KeyControl on IBM Cloud**, especifique los valores de servicio y seleccione **Añadir a instancia nueva**.

## Solicitud de HyTrust KeyControl on IBM Cloud para una instancia existente

Puede añadir el servicio de HyTrust KeyControl on {{site.data.keyword.cloud_notm}} a una instancia existente utilizando uno de los métodos siguientes:
* Desde la consola de {{site.data.keyword.vmwaresolutions_short}}, visualice la instancia para la que desea añadir el servicio, pulse **Servicios** en el panel de navegación izquierdo y pulse **Añadir**.
* Desde el catálogo de {{site.data.keyword.cloud_notm}}, seleccione **HyTrust KeyControl on IBM Cloud**, especifique los valores de servicio y seleccione **Añadir a instancia existente**.

## Configuración del servicio de HyTrust KeyControl on IBM Cloud

Al solicitar el servicio, proporcione los valores siguientes:

Especifique si desea crear un clúster de KeyControl de alta disponibilidad de dos nodos:
* Si selecciona esta opción, se despliegan dos nodos KeyControl y se crea un nuevo clúster de alta disponibilidad activo-activo. Esta opción es la predeterminada.
* Si no selecciona esta opción, se despliegan dos nodos KeyControl autónomos sin que se haya creado ningún clúster. Los nodos autónomos se pueden agrupar de forma manual o se pueden añadir a clústeres de KeyControl existentes después del despliegue.

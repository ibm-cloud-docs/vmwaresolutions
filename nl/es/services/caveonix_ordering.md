# Solicitud de Caveonix RiskForesight on IBM Cloud
{: #caveonix_ordering}

Puede solicitar el servicio Caveonix RiskForesight on {{site.data.keyword.cloud}} al pedir una instancia nueva de VMware vCenter Server con el servicio o añadiendo el servicio a la instancia existente.

## Solicitud de Caveonix RiskForesight on IBM Cloud para una nueva instancia
{: #caveonix_ordering-new}

Para solicitar una nueva instancia de VMware vCenter Server on {{site.data.keyword.cloud_notm}} con Caveonix RiskForesight on {{site.data.keyword.cloud_notm}}, seleccione **Caveonix RiskForesight on IBM Cloud** en la sección **Servicios** al solicitar la instancia desde la consola de {{site.data.keyword.vmwaresolutions_short}}.

## Solicitud de Caveonix RiskForesight on IBM Cloud para una instancia existente
{: #caveonix_ordering-existing}

Para añadir el servicio Caveonix RiskForesight on {{site.data.keyword.cloud_notm}} a una instancia existente de VMware vCenter Server on {{site.data.keyword.cloud_notm}}, visualice la instancia para la que desea añadir el servicio, pulse **Servicios** en el panel de navegación izquierdo y, a continuación, pulse **Añadir**.

## Configuración de Caveonix RiskForesight on IBM Cloud
{: #caveonix_ordering-config}

Al solicitar el servicio Caveonix RiskForesight on {{site.data.keyword.cloud_notm}}, proporcione los valores siguientes:

### Nombre de licencia
{: #caveonix_ordering-config-license-name}

Especifique un nombre exclusivo para la licencia.

### Notas de licencia
{: #caveonix_ordering-config-license-notes}

Especifique las notas para la instancia de servicio. Puede editar las notas más adelante en la página de detalles de licencia.

### Número de máquinas virtuales para licencia
{: #caveonix_ordering-config-vms}

Seleccione el número de licencias de máquina virtual (de 10 hasta un máximo de 1000 licencias de máquina virtual en incrementos de 10). Se necesitan como mínimo 10 máquinas virtuales para la gestión de licencias.

Después de solicitar el servicio Caveonix RiskForesight on {{site.data.keyword.cloud_notm}}, si desea más máquinas virtuales con licencia que las especificadas en el pedido, puede sustituir la licencia de Caveonix RiskForesight solicitando una licencia mayor, copiando la licencia en la consola de Caveonix RiskForesight y, a continuación, suprimiendo la licencia anterior de la consola de {{site.data.keyword.vmwaresolutions_short}}. Para obtener más información, consulte [Procedimiento de solicitud de licencias de Caveonix RiskForesight](/docs/services/vmwaresolutions?topic=vmware-solutions-caveonix_license_ordering#caveonix_license_ordering-procedure) y [Procedimiento de supresión de licencias de Caveonix RiskForesight](/docs/services/vmwaresolutions?topic=vmware-solutions-caveonix_license_managing#caveonix_license_managing_procedure-delete).

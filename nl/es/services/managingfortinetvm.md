# Gestión de FortiGate Virtual Appliance on IBM Cloud

## Acceso a la consola de FortiGate

Para gestionar el servicio FortiGate Virtual Appliance on {{site.data.keyword.cloud}}, debe acceder a la consola de FortiGate® de una de estas formas:
* Inicie una sesión en el cliente web de FortiOS utilizando las credenciales que encontrará en la página de detalles del servicio FortiGate Virtual Appliance on {{site.data.keyword.cloud_notm}}.
* Acceda a la consola mediante una conexión SSH utilizando las credenciales que encontrará en la página de detalles del servicio FortiGate Virtual Appliance on {{site.data.keyword.cloud_notm}}.

Para obtener más información, consulte los temas siguientes:
